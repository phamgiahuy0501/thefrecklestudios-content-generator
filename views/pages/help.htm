How to use Descripsion[0] Generate?
Step by step guide below will help you from generating HTML element to apply it to Haravan product.
  • Content
    Input contents to the Input panel.
  • Generate HTML
    Click Run button in to generate HTML.
  • Review
    Review how it works, is it exactly what you want.
  • Copy
    Copy all your work and move on to the next step.
  • Source option in Product Haravan
    Go to a specific product page that you want to change or update description. Choose source option in Edit option to open Source edit panel.
  • ?
    Do contents of the Source edit panel have any "####" ?
    No
    Yes
  • Past or Replace all
    Past all your work has been copied to Source edit panel.
    Ctrl+V or Command+V
  • Past or Replace
    Past all your work has been copied BEFORE the FIRST "####" to Source edit panel.
    Ctrl+V or Command+V
  • Save and Preview your work
    Save and close Source edit panel. Update changes product and Preview your work.
